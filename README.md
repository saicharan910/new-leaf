New Leaf Sanctuary
"Your story isn't over yet. Let us help you turn the page."

New Leaf is a high-empathy, crisis-intervention web platform designed to provide immediate hope and verified professional resources to individuals facing depression, grief, or suicidal thoughts. Built with a "Help First, Logistics Later" philosophy, the platform ensures that mental health support is just one click away.

🌟 Features
Empathy-First Design: A calming, Zen-inspired UI using a Sage Green and Earth-tone palette to reduce user anxiety.

Direct Call Tabs: One-tap buttons that immediately trigger the phone dialer for national helplines like Tele-MANAS (14416), KIRAN, and Vandrevala.

Smart Location Routing: Dynamically displays verified psychiatric and emergency centers based on the user's selected city (special focus on Tirupati, Guntur, and Vijayawada).

Voice-to-Text Entry: Integrates the Web Speech API ("Speak Your Heart") to allow users in distress to record their thoughts without typing.

Crisis Dispatch System: Automatically formats user stories into high-priority emergency reports and analyzes text for urgency keywords to alert responders.

Responsive & Secure: Fully mobile-responsive and deployed over HTTPS to ensure privacy and microphone access.

🛠️ Technology Stack
Frontend: HTML5, CSS3 (Custom Variables), JavaScript (ES6+).

Styling: Tailwind CSS architecture for a modern, clean interface.

APIs: Web Speech API (Speech-to-Text), Browser Geolocation.

Communication: tel: and mailto: protocols for direct-to-responder contact.
